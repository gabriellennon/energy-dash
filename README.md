## Install and Run
```sh
# clone the project
git clone https://github.com/gabriellennon/energy-dash.git
# install dependencies 
yarn install ou npm install

# Run the fake back-end first
npm run fake-back

# Run the fake front-end
npm run dev


# Run tests
npm run test
```
